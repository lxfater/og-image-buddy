# og-image-buddy

一个强大且易用的开放图谱（Open Graph）图片生成器，为您的网络应用提供完美的社交媒体预览图片。

## ✨ 特性

- 支持自动截图网站
- 支持自动获取logo
- 能够离线使用，无需翻墙
- 支持中文输入

未来计划支持

- api模式，批量生成图片
- 支持推文模板
- 博客图模板
